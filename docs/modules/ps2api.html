ps2api Module

Defined in: lib\User.js:19

Adds the connection capability to the module

This module provides the following classes:

  • Connection
  • Event
  • Item
  • User

This module is a rollup of the following modules:

  • ps2api-connection
    Adds the connection capability to the module
  • ps2api-event
    Adds the event emitter to the module
  • ps2api-item
    Adds the item structure to the module
  • ps2api-user
    Adds the user class to the module
